How should room type show in the status grid?

Three approaches — from lightest to most informative

A

Color-coded by type

Each room type gets its own shade of amber — Standard (light), Deluxe (medium), Suite (deep). Occupancy shown by fill. No extra text needed once you learn the legend.

C

Keep it simple — room number only

Dashboard stays a quick glance. Full room type details live on the Rooms page where staff go to assign. Staff learn the rooms quickly anyway.

Best for small properties (<20 rooms) where staff knows the inventory.
